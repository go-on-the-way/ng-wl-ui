# NgWlUi
一个基于angular的自定义UI库
## UI组件开发、编译、发布
+ 生成组件
    ng generate component <component-name> --project=ng-j2
+ 编译
    npm run build:library
+ 发布
  1.进入编译目录
    cd dist/ng-j2
  2.发布到npm
    npm publish

## 备注：将angular组件库发布到npm上，可参考[这里](https://blog.csdn.net/wanglei1991gao/article/details/85330204)
